⚠ [Erro #4] CSS: about-grid com 1 coluna em vez de 2             \
⚠ [Erro #5] CSS: gap negativo nos depoimentos                    \
✖ [Erro #6] JS: função do formulário com bug — veja o script    \
ℹ [Erro #7] HTML: link do rodapé quebrado (#section-errado)      \
